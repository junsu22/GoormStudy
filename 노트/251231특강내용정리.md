# 생성형 AI 개념 및 최신 동향 정리

## 1. 생성형 AI 정의와 개념
**생성형 AI(Generative AI)** 는 기존 데이터를 학습하여  
새로운 콘텐츠를 창작할 수 있는 인공지능 기술이다.

- 텍스트
- 이미지
- 음악
- 영상
- 코드

등 **다양한 형태의 콘텐츠 생성**이 가능하다.

---

## 2. 학습 메커니즘
생성형 AI는 방대한 양의 데이터를 학습하여  
**확률적 패턴과 규칙**을 파악한다.

- 데이터 내 구조 학습
- 확률 기반 다음 토큰 예측
- 기존 패턴을 조합하여 **새로운 데이터 생성**

---

## 3. 판별형 AI와의 차이점

### 🔹 판별형 AI
- 목적 : 분류, 예측, 판별
- 예시 : 스팸 필터, 이상 탐지, 추천 시스템
- 특징 : **정답 맞추기 중심**

### 🔹 생성형 AI
- 목적 : 새로운 콘텐츠 창작
- 예시 : 텍스트 생성, 이미지 합성, 음악 작곡
- 특징 : **창작 중심**

---

## 4. 생성형 AI 역사 타임라인

### 2014 – GAN 등장
- Ian Goodfellow가 **GAN(생성적 적대 신경망)** 발표
- 이미지 생성 분야에 혁명적 변화

### 2017 – Transformer 구조
- Google, *Attention is All You Need*
- 현대 NLP 모델의 근간

### 2018 – GPT-1
- OpenAI, 최초 GPT 공개
- 자연어 생성 모델의 시작

### 2019~2020 – GPT-2 / GPT-3
- GPT-3 (1750억 파라미터)
- 텍스트 생성 성능 비약적 향상

### 2022 – ChatGPT
- GPT-3.5 기반 대화형 AI
- 출시 1개월 만에 사용자 1억 명 돌파

### 2023~2025 – LLM 경쟁 시대
- GPT-4/5, Claude, Gemini, LLaMA 등
- 멀티모달 기능 본격 확장

---

## 5. 기존 AI vs 생성형 AI 비교

| 구분 | 생성형 AI | 판별형 AI |
|----|----|----|
| 목적 | 콘텐츠 생성 | 분석·예측 |
| 학습 방식 | GAN, Transformer | 지도/비지도/강화 |
| 대표 모델 | GPT, Gemini, Stable Diffusion | 분류기, 회귀 |
| 상호작용 | 대화형, 반복 개선 | 일방향 결과 |

## 6. 생성형 AI 대표 활용 사례

### ✏️ 텍스트 생성
- 질의응답
- 보고서 작성
- 리서치 요약

### 🎨 이미지 생성
- 텍스트 → 이미지
- 이미지 편집 및 합성

### 💻 코드 생성
- 코드 작성
- 디버깅
- 리팩토링

### 🎵 음성 / 음악 생성
- 음성 합성
- 음악 작곡

## 7. DeepResearch
- Reasoner 기반 **멀티스텝 웹 리서치**
- 복잡한 주제 정보 종합
- 기술·연구 질문에 심층 답변 제공

## 8. ChatGPT Agent
- 가상 컴퓨터 기반 자율 작업 수행
- 검색 → 분석 → 요약 → 보고서 자동화
- DeepResearch 기능과 연동
